    Developper Guide Code style

    Friday, August 5, 2016 3:24 PM

    Here are a few guidelines regarding code style.

    Automatic formating

    Artistic Style is used to uniformise indentation, padding, tabulations...

    If you use Sublime Text, there is a plugin for Artistic Style called SublimeAStyleFormatter. The plugin is configured by adding the following lines in the configuration file. Feel free to open an issue to propose changes to this default style.

    "AStyleFormatter":
{
    "options_default":
    {
        "indent": "spaces",
        "indent-spaces": 4,
        "convert-tabs": true,
        "style": "bsd",
        "indent-switches": true,
        "pad-oper": true,
        "unpad-paren": true,
        "align-pointer": "type",
    },
}
