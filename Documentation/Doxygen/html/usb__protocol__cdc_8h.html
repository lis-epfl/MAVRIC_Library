Macro Definition Documentation

#define CDC_CTRL_SIGNAL_ACTIVATE_CARRIER   (1 << 1)

Carrier control for half duplex modems. This signal corresponds to V.24 signal 105 and RS-232 signal RTS. The device ignores the value of this bit when operating in full duplex mode.

#define CDC_CTRL_SIGNAL_DTE_PRESENT   (1 << 0)

Indicates to DCE if DTE is present or not. This signal corresponds to V.24 signal 108/2 and RS-232 signal DTR.
